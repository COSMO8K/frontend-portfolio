# Frontend Developer Portfolio

This project is my personal frontend developer portfolio website.  
It is created to showcase my skills, projects and experience in web development.

## 🚀 Technologies used

- HTML5
- SCSS (Sass)
- JavaScript
- Gulp
- Git & GitHub
- Responsive Layout (Flexbox / Grid)

## 💻 Features

- Modern responsive layout
- Burger menu for mobile devices
- Clean and structured code
- Optimized styles with SCSS
- Project structure using Gulp

## 🎯 Purpose of the project

The goal of this project is to practice frontend development and create a portfolio that I can use when applying for internships and junior frontend developer positions.
